Часті запитання

Чи це безпечно для моїх даних?

Так. Скрипт працює всередині вашої Google Таблиці. Ми не маємо доступу до ваших даних після завершення налаштування. API ключі зберігаються у захищених властивостях вашого скрипта.

Що робити, якщо API Нової Пошти зміниться?

Ми будуємо код на стабільних версіях API. У разі глобальних змін, ми надаємо оновлення скрипта в рамках технічної підтримки.

Чи підходить це для магазинів в Instagram?

Ідеально! Більшість наших клієнтів — саме Instagram-магазини, які ведуть облік у таблицях і хочуть швидко масштабуватися без найму додаткових менеджерів.
